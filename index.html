• About

    Quest is a research operating system designed and implemented at Boston University over the past few years. The main web page is here.

    Quest is open source software available under the terms of the GNU General Public License v3.

  • Download

    Due to its experimental nature, Quest is only available through a git repository.

    You can download this project in either zip or tar formats.

    You can also clone the project with git by running:

    $ git clone git://github.com/QuestOS/quest
    And you can visit the GitHub page directly: QuestOS/quest
